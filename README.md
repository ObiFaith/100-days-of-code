## Day 5 | June 17, 2023

- Read on Function
  - Closure
  - Recursive Function

```javascript
let square = function (num) {return num * num};

let sq = square;
console.log(sq(4)); // 16
```

```javascript
  /* Closure in JS */
let square = (num) => {return (n) => n * num};
let sq = square;

console.log(sq(3)(4)); // 12
```
